Quick Start Guide

1. Create Your Account

Sign up for a free MeatLizard account to get started. No email required - just choose a username and password.

Create Account

2. Explore the Dashboard

Once logged in, you'll see your dashboard with access to all platform features:

  • AI Chat: Start conversations with our AI models
  • URL Shortener: Create short links with analytics
  • Pastebin: Share code and text snippets
  • Video Platform: Upload and stream videos
  • Media Import: Import content from other platforms
  • File Storage: Manage your uploaded files

3. Start Your First AI Chat

Click on "AI Chat" from your dashboard to open the chat interface. The AI will greet you and you can start asking questions or having conversations.

Try AI Chat

System Architecture

How MeatLizard Works

MeatLizard uses a unique hybrid architecture:

  • Web Interface: FastAPI-based web application for user interaction
  • Server Bot: Discord bot that orchestrates AI sessions and manages channels
  • Client Bot: Specialized bot running on Apple Silicon for AI inference
  • Local AI: llama.cpp with Metal acceleration for fast, private inference
  • End-to-End Encryption: All bot communication is encrypted for security
